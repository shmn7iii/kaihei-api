# kaihei

## 🏠 Project Structure

- Frontend: TypeScript, Astro
- Backend: Go, echo

## 🚀 Setup

Copy .env of each frontend and backend, and write them.

```bash
$ cp frontend/.env.sample frontend/.env
$ cp backend/.env.sample backend/.env
```

Then, up the servers.

```bash
$ docker compose up -d
```
